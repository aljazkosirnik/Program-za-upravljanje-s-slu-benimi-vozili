Velika farmacevtska družba v tvojem kraju ima vozni park, v katerem so avtomobili, kombiji, tovornjaki in še kaj. Do sedaj so seznam vozil vodili na roke (preko knjig), zdaj pa so se odlocili da se digitalizirajo. Zato so te najeli, da jim narediš Python program.

Vsako vozilo ima te atribute:

znamka
model
število prevoženih kilometrov
datum zadnjega servisa
Program naj omogoca uporabniku naslednje:

Ogled seznama vozil (voznega parka)
urejati število prevoženih kilometrov za vsako vozilo
urejati datum zadnjega servisa za vsako vozilo
dodati novo vozilo
izbrisati vozilo s seznama
